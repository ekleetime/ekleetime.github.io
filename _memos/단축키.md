## toad

토드(Toad) 사용법 : 단축키
오라클 관리툴로 많이 사용되고 있는 것 중에 하나가 토드인데요.
토드에서 편리하게 사용할 수 있는 단축키를 알아보겠습니다.

▶테이블 정보 상세보기

F4 : Table, View, Proc, Funct, Package를 DESC(테이블명 위에 커서를 두고 F4키)

 

▶자동완성

Ctrl+. : Table Completion (매칭되는 테이블목록 출력)

Ctrl+T : Columns Dropdown (해당테이블의 컬럼목록 표시)


▶SQL문 실행

F5 : SQL Editor내의 모든 SQL문 실행

Ctrl+Enter : 현재 커서의 SQL문 실행

F9 : SQL문 실행 후 Grid에 출력

 

▶히스토리(과거 수행 SQL 조회)

F8 : 과거에 실행한 SQL HISTORY 목록

Alt+Up : History UP

Alt+Down : History DOWN

 

▶텍스트 대/소문자 변환

CTRL+L : 텍스트를 소문자로

CTRL+U : 텍스트를 대문자로

 

▶주석처리

Ctrl+B : 주석처리

Ctrl+Shift+B : 주석해제

 

▶편집 창 전환(이동)

F6 : SQL Editor와 결과창간의 이동

F2 : SQL Editor창 전체화면 전환

Shift+F2 : Grid Output창 전체화면 전환

 

▶기타 단축키

F7 : 화면을 모두 CLEAR

Ctrl+Shift+F : 쿼리문 정렬

Ctrl+F9 : SQL문 Validate (SQL문을 수행하지 않음)




 

F1 Toad 도움말 파일의 SQL Editor 부분이 표시됩니다.

F2 전체 화면 Editor와 Editor/Results 패널 표시 장치 사이를 전환

<SHIFT>F2 전체 화면 그리드를 전환

F3 다음으로 일치하는 것 탐색

<SHIFT>F3 이전에 일치하는 것 탐색

F4 팝업 창의 테이블, 뷰, 프로시저, 함수, 또는 패키지 설명

F5 스크립트로 실행

F6 커서를 Editor와 Results 패널 사이로 전환

F7 모든 텍스트 삭제

F8 이전 SQL 문을 재호출

F9 실행문 실행

<CTRL>F9 실행(구문 분석) 없이 실행문 검사

<SHIFT>F9 커서 위치에서 현재 실행문 실행

F10 오른쪽 클릭 메뉴를 표시합니다.

F11 Script와 같은 것 실행(=F5)

F12 편집기 내용을 지정된 외부 편집기로 전달

<CTRL>A 모든 텍스트 선택

<CTRL>C 복사

<CTRL>D 프로시저 인수 표시

<CTRL>E 현재 실행문에서 Explain Plan을 실행

<CTRL>F 텍스트 검색

<CTRL>G 라인으로 이동

<CTRL>L 텍스트를 소문자로 변환

<CTRL>M Make Code Statement

<CTRL>N 이름이 지정된 SQL 문을 재호출

<CTRL>O 텍스트 파일 오픈

<CTRL>P Strip Code Statement(쓸데없는 태그 정리)

<CTRL>R 검색 및 바꾸기

<CTRL>S 파일 저장

<SHIFT><CTRL>S 파일을 다른 이름으로 저장

<CTRL>T 열 드롭다운 표시

<CTRL>U 텍스트를 대문자로 변환

<CTRL>V 붙여넣기

<CTRL>X 잘라내기

<SHIFT><CTRL>Z 마지막으로 취소한 작업 재실행

<ALT><UP> 이전 실행문 표시

<ALT><DOWN> 다음 실행문 표시

<ALT><PgUp> 이전 탭으로 이동

<ALT><PgDn> 다음 탭으로 이동

<CTRL><ALT><PgUp> 이전 결과 패널 탭으로 이동

<CTRL><ALT><PgDn> 다음 결과 패널 탭으로 이동

<CTRL><HOME> 데이터 그리드에서는 맨 위의 레코드셋으로 이동, 편집기에서는 텍스트의 첫 번째 열과 첫 번째 행으로 이동

<CTRL><END> 데이터 그리드에서는 레코드셋의 맨 끝으로 이동, 편집기에서는 텍스트의 마지막 열과 마지막 행으로 이동

<CTRL><SPACE> 코드 완성 템플릿 활성화

<CTRL><TAB> MDI Child 창의 콜렉션 순환

<CTRL><ENTER> 커서 위치에서 현재 SQL 실행

<CTRL>. (마침표) 테이블 이름 자동완성
